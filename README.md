## Project Overview
This project analyzes customer behavior and revenue performance to identify key insights related to project success, funding trends, and customer contribution patterns.

The dashboard helps stakeholders understand performance metrics and identify factors influencing revenue growth.

## Problem Statement
Businesses need data-driven insights to understand customer contribution patterns, funding performance, and project success factors.

Understanding these patterns helps improve decision-making and optimize revenue strategies.

## Tools Used
Power BI  
Excel  
sql

## Key Insights
• Majority of projects fall under successful category  
• Medium goal range projects show higher success rate  
• Product Design and Technology categories perform strongly  
• Revenue trends show peak performance during growth years  
• Backer contribution significantly influences project success

## Dashboard Features
• KPI cards (Total amount raised, total backers)
• Success rate analysis
• Category performance insights
• Year-wise trend analysis
• Location-based analysis
• Goal range performance insights

## Business Impact
The insights help stakeholders understand customer contribution patterns and revenue trends. This supports better decision-making for project planning, goal setting, and revenue optimization.
